How to view the source code of a web page in Chrome

In this example, I will demonstrate how to view Amazon's source code while searching for a hidden duck. Follow these steps below:
  1. Go to Amazon's homepage
  2. Right click on the page (or if you are using a Mac, ctrl + click)
  3. Select "inspect" at the bottom of the menu that appears
  4. You should open up to the elements panel. Scroll down through the HTML. You will find the meowing duck in green toward the bottom of the home page HTML.

Using Dev Tools, you can:

  • Change the color of your font or background
  • Go to the console tab and look for/fix errors in your CSS
  • Alter your layout so it can be compatible on different sized decives

Metaphor:

Using Dev Tools is like getting an X-Ray; you can see the structure of your bones and identify what's broken, just as a web developer can use Dev Tools to identify errors in the HTML/CSS/Java Script. However, this metaphor falls short in that in Dev Tools you can tinker with the site directly, but an X-Ray does not allow a doctor to touch the bones directly. The doctor would have to do surgery (or let the bones heal on their own) to resolve the issue.
